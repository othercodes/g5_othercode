# tpl_othercode

otherCode Joomla Template developed with Gantry 5

![template_preview.png](template_preview.png)

## Build Package

Run the following command:

```bash
sh ./tpl_othercode.sh
```

## Configuration

Access to template configuration in: **Extensions > Template > Styles > [Template Name]**
